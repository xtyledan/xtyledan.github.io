# Tyler Daniels - Software Engineer Portfolio

A modern, responsive single-page portfolio website showcasing my experience, projects, education, and skills.

## Features

- **Single-page design** - All content consolidated into one page for better user experience
- **Responsive navigation** - Sticky navigation with smooth scrolling between sections
- **Modern UI/UX** - Clean, professional design with consistent styling
- **Mobile-friendly** - Optimized for all device sizes
- **Performance optimized** - Efficient loading with preloaded resources

## Sections

1. **Home** - Hero section with introduction and call-to-action buttons
2. **Experience & Projects** - Work history and personal projects
3. **Education & Skills** - Academic background and technical skills
4. **Clubs** - Leadership and organizational involvement
5. **Contact** - Professional social media links

## Technical Details

- **HTML5** - Semantic markup with proper accessibility
- **CSS3** - Modern styling with CSS custom properties and flexbox/grid layouts
- **JavaScript** - Vanilla JS for smooth scrolling and mobile navigation
- **Font Awesome** - Icon library for visual elements
- **Google Fonts** - Typography optimization

## Recent Improvements

- ✅ Converted from multi-page to single-page layout

- ✅ Improved navigation with smooth scrolling and active state management
- ✅ Enhanced visual consistency and spacing between sections

## Usage

Simply open `index.html` in a web browser to view the portfolio. The navigation will smoothly scroll to each section, and the active navigation item will be highlighted based on the current scroll position.
